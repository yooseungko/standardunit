## 🔄 수정 방법

대표 항목을 변경하려면:

1. **관리자 페이지** → **표준 단가 관리** 이동
2. 해당 항목의 **수정** 버튼 클릭
3. **대표 항목 등급** 필드에서 `기본` 선택
4. **저장** 버튼 클릭

> ⚠️ 주의: 대표 항목이 없으면 견적서 생성이 불가능합니다.
